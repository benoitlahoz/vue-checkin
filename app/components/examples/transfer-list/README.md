# Transfer List Example

This example demonstrates a transfer list component that reads a CSV file with a `name` header, allows you to add names to a transfer list, and applies a transformation to split the name into `firstname` and `lastname` using the transform plugin.

## Features
- Upload a CSV file with a `name` column
- Add names to the transfer list
- Apply transformation to split names into firstname/lastname

## Usage
1. Select a CSV file with a `name` header (e.g. `name\nJohn Doe\nJane Smith`)
2. Add names to the transfer list
3. Click "Apply Name Transform" to split names
