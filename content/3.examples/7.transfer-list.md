An interactive transfer list example that lets you move CSV headers between panels.

::example-loader{:component="TransferList"}
::

## Project Structure

```
transfer-list/
├── index.ts           # Types, injection key, exports
├── fixtures.ts        # Mocked CSV data
├── useCsv.ts          # CSV utilities
├── useTransferList.ts # Transfer logic and state management passed as desk's context
├── TransferList.vue   # Main transfer list component
├── Transferable.vue   # Transferable item component
```

## Usage

1. View available headers from the mocked CSV
2. Transfer headers to the right panel using the "Transfer" button
3. Retrieve headers back to the left panel if needed
4. Download the transferred data as a CSV file

## Example CSV

```
name,age,city
John Doe,32,Paris
Jane Smith,28,Lyon
Alice Cooper,40,Bordeaux
```

## Result

After transferring headers, the data is displayed and can be downloaded as a CSV file.
