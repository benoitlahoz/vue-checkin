VueCheckIn can be installed in any Vue 3 or Nuxt 3 project using your preferred package manager.

## Package Manager Installation

::steps
### Install the package

Install VueCheckIn using your preferred package manager:

::code-group
```bash [npm]
npm install vue-checkin
```

```bash [yarn]
yarn add vue-checkin
```

```bash [pnpm]
pnpm add vue-checkin
```

```bash [bun]
bun add vue-checkin
```
::

### Import the composable

Import `useCheckIn` in your Vue components:

```vue
<script setup lang="ts">
import { useCheckIn } from 'vue-checkin';

// Your component logic
</script>
```

### Start building

You're ready to go! Check out the [Examples](/examples) to see VueCheckIn in action.
::

## Requirements

::callout{icon="i-lucide-info"}
VueCheckIn requires Vue 3.3+ or Nuxt 3+ for proper TypeScript support and composition API features.
::

### Vue 3 Projects

```json [package.json]
{
  "dependencies": {
    "vue": "^3.3.0",
    "vue-checkin": "latest"
  }
}
```

### Nuxt 3 Projects

```json [package.json]
{
  "dependencies": {
    "nuxt": "^3.0.0",
    "vue-checkin": "latest"
  }
}
```

## TypeScript Support

VueCheckIn is written in TypeScript and provides full type definitions out of the box. No additional type packages are needed.

```ts
import { useCheckIn } from 'vue-checkin';
import type { CheckInDesk, CheckInItem } from 'vue-checkin';

interface MyData {
  label: string;
  value: number;
}

const { createDesk } = useCheckIn<MyData>();
const { desk } = createDesk('myDesk');

// Fully typed desk and items
const items: CheckInItem<MyData>[] = desk.getAll();
```

## Using Plugins

VueCheckIn includes built-in plugins that you can import and use:

```ts
import { useCheckIn } from 'vue-checkin';
import { 
  activeItemPlugin, 
  historyPlugin,  
} from 'vue-checkin';

const { createDesk } = useCheckIn();
const { desk, context } = createDesk('myDesk', {
  plugins: [
    activeItemPlugin(),
    historyPlugin({ maxSize: 50 }),
  ]
});
```
